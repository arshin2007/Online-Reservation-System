# Online Reservation System

This is a Java-based Online Reservation System project that allows users to book and manage reservations.

## Features
- User login system
- Reservation booking
- Cancellation of bookings
- Simple console-based interface

## Technologies Used
- Java

## How to Run
1. Compile the program:
   javac OnResSys.java

2. Run the program:
   java OnResSys
